# André Larsen Portfolio

This repository contains a single-page portfolio landing page for André Larsen.

## Contents

- `index.html` — interactive landing page with ASCII text, animated background, particle effects, and navigation links.
- `IMG_5937.JPG` — background image used by the page.

## Usage

Open `index.html` in a browser to view the portfolio page.

## Notes

- The page is currently a static HTML file with inline CSS and JavaScript.
- Navigation links point to `works.html` and mail/Instagram links.
